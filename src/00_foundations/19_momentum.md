# Exercise 02: SGD with Momentum

Compare plain SGD against SGD with momentum on the function
`f(x) = x ** 4 - 3 * x ** 2 + 2`.

Run both for 30 iterations starting at `x = 2.0`, lr `0.01`. Use
`momentum=0.9` for the momentum variant.

Print the final `x` from each. Momentum should reach a minimum faster.
